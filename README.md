## Features

- AI-powered weather responses
- Calculator tool support
- Dynamic tool execution
- React chat interface
- Express backend
- Gemini API integration

- ## Tech Stack

Frontend:
- React
- Axios

Backend:
- Node.js
- Express

AI:
- Gemini API

External APIs:
- WeatherAPI

## Installation

### Backend

cd backend
npm install
node src/server.js

### Frontend

cd frontend
npm install
npm run dev
